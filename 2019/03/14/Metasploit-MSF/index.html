Metasploit(MSF)入门

发表于 2019-03-14 | 分类于 学习笔记 |

综合利用工具，集成了上千种漏洞利用方式，操作简单易上手。

kali集成了Metasploit框架，依赖于postgresql数据库

  1. 先启动数据库

  2. 启动MSF

  3. 利用MSF生成木马获取meterpreter反弹shell

    -p 选用什么payload
    lhost 监听ip
    lport 监听端口
    -f 生成文件的类型

  4. 投放木马到目标机上

  5. 启用MSF，具体操作如下：

    1. 加载expuse

    2. 加载payload（要和生成木马选用的payload一致）

    3. 查看哪些参数是需要输入的

    4. 运行

    5. 诱导用户点击我们的木马程序

    6. 等待用户上线，就能得到meterpreter,

    7. 执行shell 进到被攻击者电脑的终端

    8. 添加添加用户 开机3389 连接桌面(
