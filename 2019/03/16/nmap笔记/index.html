更新于: 2020-09-05 | 分类于 渗透测试 |

一、介绍

nmap (network mapper) 最早是Linux中扫描和嗅探工具，网络连接端扫描工具，主要是探测目标机器上开启的服务、安全性问题、主机存活等等

二、基本功能

  1. 主机探测；（是否开机）
  2. 服务版本检测；（端口）
  3. 操作系统检测；（OS类型：NT/Linux）
  4. 网络路由跟踪；
  5. 扫描目标机器上的安全漏洞。（脚本引擎NSE）
